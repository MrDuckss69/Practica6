Lenguajes Frontend

Los lenguajes frontend son aquellos que se ejecutan del lado del cliente y controlan el contenido y la interacción que los usuarios ven en sus navegadores.

  • HTML
  • CSS
  • JavaScript

Frameworks Frontend

Existen varios frameworks frontend que ayudan a simplificar el desarrollo web:

  • React
  • Angular
  • Vue.js

Bibliotecas y Herramientas

Además de los lenguajes y frameworks, hay bibliotecas y herramientas que pueden ser útiles, como Bootstrap para el diseño o jQuery para la manipulación del DOM.
