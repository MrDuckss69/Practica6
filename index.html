¿Qué es HTML?

HTML (HyperText Markup Language) es el lenguaje estándar para crear la estructura y el contenido de las páginas web. Define los elementos como encabezados, párrafos, enlaces, imágenes, y otros tipos de contenido multimedia.

¿Qué es CSS?

CSS (Cascading Style Sheets) es el lenguaje usado para describir la presentación visual de un documento HTML. Permite controlar el diseño, colores, fuentes y la disposición del contenido en una página web.

¿Qué es JavaScript?

JavaScript es un lenguaje de programación que se utiliza para crear interactividad y dinamismo en las páginas web. Permite la creación de funciones como la validación de formularios, animaciones y la interacción con APIs.
